Brainfucked Changelog

Im Folgenden sind alle Veränderungen gegenüber Vorgängerversion von Brainfucked aufgelistet.

Version 1.0.0 (01.05.2005)

  • Code-Optimierung noch weiter verbessert
  • Brainfuckprogramme haben jetzt ein 44.000 Bytes großes Array (macht -o überflüssig)
  • Verwendung von while statt von do-while-Schleifen
  • Parameter -n und "native mode" hinzugefügt
  • Unterstützung von für Unix geschriebene Brainfuckprogramme
  • Neue Fehlerbehandlung
  • Neue Beispielprogramme

Version 0.1.0 (23.02.2005)

  • Komplett überarbeitet und neu geschrieben
  • Starke Reduzierung der Größe (jetzt 779 Byte, war 1067 Byte)
  • Code-Optimierung um add und sub erweitert / teilweise neue Opcodes
  • Aus Kompatibilitätsgründen standardmäßg 9999 Byte großes Array
  • Parameter -o hinzugefügt für optimierte Arraygröße wie bisher
  • BFD Xtreme Edition 0.0.1 testweise hinzugefügt
  • Überarbeitete demo.b und liesmich.txt, neue Datei readme.txt

Version 0.0.2 (25.01.2002)

  • Beendet sich mit Returncode
  • Läuft sowohl unter Windows 9x als auch Windows NT/2000/XP, sowie MS-DOS
  • Kleinere Verbesserungen
