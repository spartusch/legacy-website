Herzlich willkommen

Willkommen auf der Homepage von Stefan Partusch. Schön, dass Sie hierher gefunden haben!
Was Sie hier genau finden können, verrät Ihnen das Inhaltsverzeichnis. Der Schwerpunkt liegt auf den von mir erstellten Programmen, die hier kostenlos angeboten werden. Viele davon sind "Open Source", also Programme bei denen der Quellcode der Allgemeinheit zur freien Verfügung steht. Alles auf dieser Website ist mindestens Freeware und darf frei verwendet werden. Die Benutzung der Programme erfolgt immer auf eigene Gefahr. Wenn Sie möchten, können Sie mir auch gerne eine E-Mail schreiben. Die E-Mail-Adresse dazu ist auf der Über-Seite zu finden.
Nun aber viel Spaß beim Stöbern auf meiner Website!
