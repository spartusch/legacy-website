DevCont - Changelog

The following lists all changes and improvements to all versions of DevCont.

Version 1.0.1 (2002-06-27)

  • Small bugfix (settings weren't loaded)

Version 1.0.0 (DevCont - 2002-06-23)

  • New name ("DevCont" instead of "TCP/UDP Connector")
  • Position and size of the window is saved
  • Color schemes
  • Made the status bar clickable
  • Some minor modifications

Version 1.2.2 (2002-05-21)

  • Knows 20 "well known ports" now
  • Small bug in binary mode fixed
  • Some minor modifications

Version 1.2.1

  • Some minor modifications

Version 1.2.0 (2002-04-05)

  • New advanced mode added!
  • Settings are saved upon program termination
  • Bugfix: now works on Windows 98
  • Some improvements

Version 1.1.0

  • New experimental binary mode introduced
  • Statusbar: status, connection, mode, protocol, sent/received bytes
  • Many, many improvements

Version 1.0.2 (2002-02-14)

  • Received Unix and Mac linebreaks are shown correctly
  • Modified context menu of the right mouse-key
  • Minor improvements

Version 1.0.1 (TCP/UDP Connector - 2002-02-05)

  • First release
