Japtra - Japanese Trainer

Japtra asks you Japanese words and phrases to assist you while learning the language. Its vocabulary is user-defined, extendable and editable using a simple text-editor like Windows Notepad. Thus Japtra allows you to adapt it to any course. Furthermore Japtra is capable of managing several courses. Therefore it's perfectly suitable for usage together with a Japanese course in school or university.
It even features a complete implementation of kana, the Japanese syllabaries hiragana and katakana. This makes Japtra independent from any external Japanese language support, allowing it to be run on any Windows operating system.
When actually using Japtra, it offers you four different modes to ask the words (Hiragana/English and Katakana/English as well as English/Romaji and Romaji/English). In addition Japtra allows you to dynamically create nice vocabulary lists for each course.

Japtra asking Japanese words

A short summary of some of the features of Japtra:

  • Doesn't need any external Japanese language support
  • Vocabulary and courses are extendable and editable
  • Creates HTML-vocabulary lists (e.g. for printing)
  • Four modes: Hiragana/English, Katakana/English, English/Romaji, Romaji/English
  • Detailed error analysis after each run

The current version of Japtra is Japtra Version 1.0.1. See the changelog for detailed descriptions of changes and improvements to older versions. Please pay attention to the license agreements! Information on the usage of Japtra is available in its documentation.
