Brainfucked - Changelog

The following lists all changes and improvements to all versions of Brainfucked.

Version 1.0.0 (2005-05-01)

  • Code optimization improved and extended
  • Brainfuck programs now have an array of 44,000 bytes available (obsoletes -o)
  • Use of while-loops instead of do-while-loops
  • Parameter -n and "native mode" added
  • Support of brainfuck programs written for Unix
  • New handling of errors and warnings
  • New examples to demonstrate brainfuck

Version 0.1.0 (2005-02-23)

  • Completely rewritten
  • Much smaller size (now 779 bytes, was 1067 bytes)
  • Code optimization with add and sub enhanced / some new opcodes
  • Default array size now 9999 bytes, greater compatibility
  • Argument -o added to optimize array size like in older versions
  • BFD Xtreme Edition 0.0.1 added
  • Revised demo.b and liesmich.txt, new file readme.txt

Version 0.0.2 (2002-01-25)

  • Terminates with a return code
  • Runs on Windows 9x and Windows NT/2k/XP and on MS-DOS
  • Minor improvements

Version 0.0.1

  • First release
